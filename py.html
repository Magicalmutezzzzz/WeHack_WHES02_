Learn Python

What is Python?

Python is a high-level, interpreted programming language known for its simplicity and readability. It is widely used in web development, data science, automation, and artificial intelligence.

Why Learn Python?

  • Easy to learn and use.
  • Great for beginners and professionals.
  • Used in various fields such as web development, data science, and AI.
  • Large community and extensive libraries.

Python Syntax

Python has a simple and readable syntax. Below is a basic example:

print("Hello, World!")

Python Basics

Variables

Python variables do not require explicit declaration.

name = "Alice"
age = 25
print(f"Name: {name}, Age: {age}")

Functions

Functions in Python allow you to reuse code.

def add(a, b):
    return a + b

print("Sum:", add(5, 3))

Classes and Objects

Python supports object-oriented programming.

class Car:
    def __init__(self, brand, year):
        self.brand = brand
        self.year = year
    
    def display(self):
        print(f"Brand: {self.brand}, Year: {self.year}")

my_car = Car("Toyota", 2022)
my_car.display()

Advanced Python Concepts

List Comprehensions

squares = [x**2 for x in range(10)]
print(squares)

Lambda Functions

square = lambda x: x**2
print(square(5))
