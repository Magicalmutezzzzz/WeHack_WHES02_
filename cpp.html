Learn C++

What is C++?

C++ is a powerful general-purpose programming language that is widely used for system/software development, game development, and competitive programming.

Why Learn C++?

  • It is widely used in high-performance applications.
  • Supports object-oriented programming.
  • Provides low-level memory manipulation capabilities.
  • Great for competitive programming and algorithm development.

C++ Syntax

C++ syntax is structured and efficient. Below is a basic example:

#include 
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}

C++ Basics

Variables

C++ supports various data types such as int, float, double, and string.

int age = 25;
double pi = 3.1415;
string name = "Alice";
cout << "Name: " << name << ", Age: " << age << endl;

Functions

Functions in C++ allow you to modularize your code:

int add(int a, int b) {
    return a + b;
}

int main() {
    cout << "Sum: " << add(5, 3) << endl;
    return 0;
}

Classes and Objects

C++ supports object-oriented programming:

class Car {
public:
    string brand;
    int year;

    void display() {
        cout << "Brand: " << brand << ", Year: " << year << endl;
    }
};

int main() {
    Car myCar;
    myCar.brand = "Toyota";
    myCar.year = 2022;
    myCar.display();
    return 0;
}

Advanced C++ Concepts

STL (Standard Template Library)

  • Vectors
  • Queues & Stacks
  • Maps & Sets
  • Algorithms & Iterators

Pointers

Pointers allow direct memory access:

int x = 10;
int* ptr = &x;
cout << "Value: " << *ptr << endl;
