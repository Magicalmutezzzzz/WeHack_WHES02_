Learn JavaScript

What is JavaScript?

JavaScript is a versatile programming language that allows developers to create dynamic and interactive web pages. It is one of the core technologies of the web alongside HTML and CSS.

Why Learn JavaScript?

  • It is the most popular programming language for web development.
  • It enables interactive and dynamic user experiences.
  • Used in front-end and back-end development (Node.js).
  • Huge job market and career opportunities.

JavaScript Syntax

JavaScript syntax is simple and easy to learn. Below is a basic example:

// Display an alert message
alert('Hello, World!');

// Logging to console
console.log('This is JavaScript');

// Declaring variables
let name = 'John';
const age = 25;
console.log(name + ' is ' + age + ' years old');

JavaScript Basics

Variables

JavaScript has three ways to declare variables: var, let, and const.

let city = "New York";
const country = "USA";
console.log(city + ", " + country);

Functions

Functions allow you to reuse code. Here's an example:

function greet(name) {
    return "Hello, " + name + "!";
}
console.log(greet("Alice"));
